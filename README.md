# Simple ATM Simulation

## Description
A Java console-based ATM simulation that allows users to perform basic banking operations such as checking the account balance, depositing money, withdrawing money, and exiting the application.

## Features
- Check account balance
- Deposit money
- Withdraw money
- Exit the program
- Menu-driven interface
- Input validation for withdrawals

## Technologies Used
- Java
- JDK
- VS Code

## How to Run

Compile the program:

```bash
javac ATMSimulation.java
```

Run the program:

```bash
java ATMSimulation
```
